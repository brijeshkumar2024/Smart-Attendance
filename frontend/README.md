# Smart Attendance Frontend

React + Vite client for the Smart Attendance System.

## Environment

Create `frontend/.env` using `frontend/.env.example`.

Required:
- `VITE_API_BASE_URL` (example: `http://localhost:5000`)

Optional:
- `VITE_SOCKET_URL` (defaults to the API host)

## Local Development

```bash
npm install
npm run dev
```

Default local URL: `http://localhost:5173`

## Build

```bash
npm run build
npm run preview
```
